# Old Architecture Index

**Created:** 2026-07-14 (Phase 0 of new generator architecture)
**Purpose:** Complete inventory of old system files for safe retirement once the new architecture is verified.
**See also:** `docs/roadmap/generator-architecture/README.md` (Phase 9 — retirement checklist)

## When to Remove

Each file listed here should only be removed after:
1. The new generator architecture (`src/`) produces equivalent output
2. Validation score ≥ 80 for all 20+ industry profiles
3. Side-by-side comparison is documented
4. This file has been reviewed as the removal checklist

## Safety Checklist

Before removing any file, verify:

- [ ] New system generates a complete site for each industry profile
- [ ] Generated sites pass validation (score ≥ 80)
- [ ] No runtime imports cross from new code to old code
- [ ] `grep -r "from '@/components/" src/` returns nothing
- [ ] `grep -r "from '@/lib/" src/` returns only logger, utils, env
- [ ] `grep -r "from '@/app/" src/` returns nothing
- [ ] Old system is recoverable from git tag `old-architecture`
